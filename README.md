# CUDA-enabled Deep Learning Image

Contents
--------

Installed in this image are:

- Caffe + CUDNN
- Theano (no CUDNN)
- Keras
- Lasagne
- Scikit-learn
- Jupyter notebook.

The github repo and docker image are both `henryzlo/deepdock`

Requirements
------------

You will need CUDA 7.5 installed and a GPU.
